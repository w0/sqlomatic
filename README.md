# SQL-O-MATIC

Quickly convert No-Intro datafiles to SQLite databases. 

# Installation

Clone this repo

`git clone https://github.com/w0/sqlomatic.git`

Install via `go install`

`go install ./cmd/sqlomatic`

# Quick Start

Convert a No-Intro datafile to SQLite. Output to the current working directory

```bash
sqlomatic -datafile ~/Desktop/Datafile.dat
```
